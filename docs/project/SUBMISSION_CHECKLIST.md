## Common Errors to Catch

### Before Each Submission:
- [ ] School name is correct throughout (not copy-pasted from another app)
- [ ] No references to professors you haven't contacted
- [ ] Research Direction matches program strengths
- [ ] URLs are complete (not truncated)
- [ ] File uploads are correct versions (not old drafts)
- [ ] Word count within limits (if specified)
- [ ] Date/signature added (if required)

### Red Flags:
- âŒ "I'm excited to work with Professor X" (if you haven't contacted them)
- âŒ Wrong school name anywhere in documents
- âŒ Generic statements like "your excellent program"
- âŒ Broken GitHub links
- âŒ Referencing research areas the school doesn't have

## Emergency Contact Info

**Technical Issues:**
- GitHub down? Check status.github.com
- Can't access research.md? Have backup PDF ready

**Application Portal Issues:**
- Have portal help desk contact for each school
- Screenshot error messages
- Contact admissions coordinator if urgent

**Recommender Issues:**
- Have backup recommender identified
- Send polite reminders 1 week before deadline
- Provide all materials (research.md, CV, SOP draft)
